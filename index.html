FriCAS

FriCAS is a computer algebra system initially forked from Axiom (see Axiom-Wiki).

FriCAS is currently officially developed at Sourceforge. http://fricas.sourceforge.net/

This fricas-svn repository at github is a live mirror of the complete official FriCAS SVN repository.

Whenever someone commits at the sourceforge repository, a commit email triggers an update of the fricas-svn GitHub repository.

For more information of how to use this mirror in order to get the full power of GIT and still being able to commit back to the SVN repository, look at to https://sites.google.com/site/hemmecke/fricas-svn.

This mirror is an offer to developers who prefer git over svn.
